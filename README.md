# Lineage 2M Crystal Ledger

Мини-приложение для учета кристаллов в Lineage 2M:

- приход кристаллов
- расход кристаллов
- покупка кристаллов за реальные деньги
- темный анимированный интерфейс
- хранение данных в SQLite

## Запуск

```bash
npm start
```

После запуска откройте:у 

```text
http://127.0.0.1:3000
```

## Демо-данные

По умолчанию приложение не загружает демо-данные.

Если нужно вручную включить стартовые записи, запустите так:

```bash
SEED_DEMO_DATA=true npm start
```

## Структура

- `server.js` - HTTP сервер и API
- `public/` - интерфейс
- `data/ledger.db` - база данных SQLite
